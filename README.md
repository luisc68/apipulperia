# Api Pulperia

APi for pulperia App

## Requirements

- Node 18.x

## Dependencies

- Express JS
- prisma

## Commands

### Installation

bash

 npm install
 yarn intall


### Run

bash

npm run start ## start compiled source on dis/ folder
npm run dev ## watch dev mode with nodemon
yarn start ## start compiled source on dis/ folder
yarn dev ## watch dev mode with nodemon
